Dear all,

It is been a while, but we are very happy to announce the 0.2.3
release of `coq-lsp`.

The 0.2.x series has seen the merge of `serlib`, so `coq-lsp` doesn't
depend on SerAPI anymore. We recommend all SerAPI users to migrate to
`coq-lsp`, as of today, `coq-lsp` + `petanque` offer much improved
functionality.

SerAPI itself has been put in archival mode, we'd like to thank all
the contributors to it!
